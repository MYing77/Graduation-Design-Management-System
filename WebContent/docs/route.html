路由配置(Route)
主要提供路由的设置，读取功能。有了它，就能很轻易地实现单页面应用程序啦~~
注：要实现页面的跳转，路由必须配置正确，是与页面模板绑定的。然后输入对应的hash地址，会读取对应的页面模板进行渲染
方法(Method)
名称 参数/返回值 用法 描述
setRoutes

{Object} options

描述：options 参考 参数(options)

返回：route对象

 route.setRoutes(options) 必须调用该方法配置路由信息才能正常工作。
getRoutes

无

描述：根据设定的config.name读取路由列表

默认值：config.name [KITADMINROUTE]

返回：路由列表

 route.getRoutes(); 获取已配置的路由列表.
getRoute

{String}hash(hash地址)

返回：返回对应地址的路由信息

 route.getRoute(hash); 获取单个路由地址信息
params

返回类型：{Object}

返回：当前页面地址上的参数

 route.params();

获取当前页面地址上的参数,参数不存在则返回null

// http://localhost:8000/#/docs/route?keywork=xx&name=Van
var params = route.params();
// result params is: {keywork:'xx',name:'Van'}
参数(options)
名称 类型 默认值 描述
name

{String}

 'KITADMINROUTE' 路由名称，名称唯一
routes

{Array}

 []

路由地址列表

[{
  path: '/user/table',                       //路由地址
  component: '/components/table/teble.html', //模板组件地址
  name: 'Table'                              //显示名称
},
//...
]
结语(End)
