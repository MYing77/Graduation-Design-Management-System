左侧菜单(Menu)
该组件提供左侧导航功能.
方法(Method)
名称 参数 用法 描述
set

{Object} options

描述： options 参考 参数(options)

 menu.set(options) 配置menu,所设置的参数将会覆盖默认设置
removeCache

{String} key

描述：菜单的缓存key

默认值：options.cacheKey [KITADMINMENU]

 menu.removeCache(key); 清除缓存的远程数据.
render

无

 menu.render(); 渲染menu菜单
参数(options)
名称 类型 默认值 描述
elem

{String}

 undefined 配置menu渲染的目标容器
onClicked

{Function}

 undefined

菜单点击后触发的回调函数

menu.set({
  // ... 其他的一些配置
  onClicked:function(obj){
    console.log(obj.elem); //点击的目标jquery对象
    console.log(obj.hasChild); //是否有子菜单
    console.log(obj.data); //返回的数据
    console.log(obj.data.href);//当前点击的a标签的href属性值
  }
})
dynamicRender

{Boolean}

 false

是否动态渲染 --如果此参数为true,则需要设置elem,还需要设置data或者remote参数

data

{Array}

 []

本地数据集，具体数据结构请参考：数据结构(data)
remote

{Object}

 
{
  url: undefined, //远程地址
  method: 'get'   //请求方式
}

要使远程加载数据时，先要设置dynamicRender为true

此配置还可以设置其他参数，请参考axios的request配置

axios文档地址： https://www.kancloud.cn/yunye/axios/234845

cached

{Boolean}

 false

是否开启缓存？,注：只缓存远程加载的数据。
cacheKey

{String}

 KITADMINMENU

缓存key
isJump

{Boolean}

 true

是否拦截a标签的直接跳转页面
onlyOne

{Boolean}

 true

只展开一级菜单
数据结构(data)
名称 类型 描述
id

{Number}

 ID,唯一标识
title

{String}

 显示的标题
path

{String}

 地址，(a标签的href属性值)
icon

{String}

 图标，当前只支持layui图标值
pid

{Number}

 父级ID(重要)，一级菜单pid为0
open

{Boolean}

 是否展开(该值只对拥有子菜单的菜单有效)
blank

{Boolean}

 跳转到外链
children

{Array}

 子菜单的数据列表(值参考以上字段)[递归]
